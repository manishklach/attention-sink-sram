# Research Roadmap

This repository is intended as an architectural platform, not a finished answer.

## Near-term directions

- hardware scheduler integration
- compiler-generated DMA scheduling
- orchestration-aware accelerator interfaces
- bounded-buffer inference runtimes

## Distributed directions

- SmartNIC or DPU-assisted orchestration
- CXL-aware residency contracts
- topology-aware graph execution
- distributed replay coordination

## Runtime and compiler directions

- richer orchestration IR semantics
- legality checking for adaptive plans
- policy synthesis from workload traces
- cross-region residency forecasting

## Evaluation directions

- stronger trace ingestion from real runtimes
- more explicit sensitivity studies
- workload-derived topology placement comparisons
- hybrid deterministic/reactive serving models
