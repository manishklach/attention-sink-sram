# Economic Modeling

The simulator includes a simple infrastructure-cost view for distributed inference clusters.

## Cost dimensions

- bandwidth cost
- memory cost
- fabric cost
- pooled-memory efficiency
- throughput per dollar

## Why include economics

Cluster-level orchestration decisions are not only about latency. They also affect how efficiently expensive SRAM, HBM, links, and pooled memory are used. The economic panel is meant to illustrate that wider systems view.
