# Runtime Integration

This repository explores how orchestration ideas may connect to a plausible inference runtime stack.

## Stack layers

- compiler
- orchestration runtime
- scheduler
- DMA engine
- residency manager
- topology manager
- fabric
- memory tiers
- kernels

## What the integration view is for

The integration panel is meant to connect the memory and compiler concepts to a future systems architecture, not to claim a complete production design.
